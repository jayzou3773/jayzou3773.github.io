# jayzou3773.github.io

Personal academic homepage of Jay (Xinkai) Zou, built with [Hugo](https://gohugo.io/) and [PaperModX](https://github.com/GindaChen/hugo-PaperModX).

## Local Development

```bash
# Install Hugo (macOS)
brew install hugo

# Clone with submodules
git clone --recurse-submodules https://github.com/jayzou3773/jayzou3773.github.io.git
cd jayzou3773.github.io

# Start dev server
hugo server
```

Open http://localhost:1313 in your browser.

## Deployment

Automatically deployed via GitHub Actions on push to `main`.
